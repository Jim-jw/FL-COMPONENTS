# FlEchart echart组件

- 使用方法
```
<FlEchart :options="options"/>
```

### FlEchart Attributes
| 参数 | 说明 |	类型 | 默认值 |
|:-|:-:|-:|-:|
| options | 渲染echarts图表的值 | object | {} |
| isResize | 图表是否监听窗口变化而变化 | boolean | false |
|||||

### FlEchart Events
| 事件名称 | 说明 |	回调参数 |
|:-|:-:|-:|
| click | 点击触发 | 包含点击图形的数据信息的对象 |
||||
